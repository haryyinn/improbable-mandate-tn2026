## Project Structure

```
Politics/
├── data/
│   ├── raw/         ← place downloaded CSVs here
│   └── processed/   ← auto-generated by scripts
├── outputs/
│   ├── figures/     ← all charts saved here automatically
│   └── The_Improbable_Mandate_TVK_TN2026.docx  ← FINAL PAPER
├── scripts/
│   ├── 01_collect_eci_data.py    ← data fetching + download guidance
│   ├── 02_clean_and_merge.py     ← clean & unify datasets
│   ├── 03_sir_analysis.py        ← SIR voter roll analysis by age group
│   ├── 04_historical_analysis.py ← 2001-2026 trends, FPTP, margins
│   ├── 05_monte_carlo.py         ← Monte Carlo (N=50,000) + surprisal
│   ├── 06_bribe_analysis.py      ← EC cash seizure proxy analysis
│   └── 07_generate_paper.py      ← assembles the Word document
├── run_all.py        ← ONE COMMAND: runs everything, outputs .docx
└── requirements.txt
```

---

## Quick Start

```bash
# 1. Install dependencies (one time)
pip3 install -r requirements.txt

# 2. Run everything — generates all figures + the Word document
python3 run_all.py
```

The paper will appear at:
`outputs/The_Improbable_Mandate_TVK_TN2026.docx`

## Data You Need to Download Manually

The scripts generate synthetic placeholder data automatically so the paper
always compiles. Replace with real data for final results:

| File | Source | Save As |
|---|---|---|
| Historical TN results 2001–2021 | [TCPD Lok Dhaba](https://tcpd.ashoka.edu.in/lok-dhaba/) — filter TN, download CSV per year | `data/raw/tn_results_YYYY.csv` |
| 2026 results | [ECI](https://results.eci.gov.in/) — auto-fetched or manual | `data/raw/tn2026_constituency_results.csv` |
| SIR voter roll stats (age-wise) | [CEO Tamil Nadu](https://www.elections.tn.gov.in/) → Electoral Roll Stats | `data/raw/tn_sir_2026.csv` |
| EC seizure data | CEO Tamil Nadu press releases / [ADR](https://adrindia.org/) | `data/raw/tn_cash_seizures_2026.csv` |
